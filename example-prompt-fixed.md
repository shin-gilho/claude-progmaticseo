## 주요 변경사항

### ✅ 해결된 문제들

1. **`{{키워드}}` → `{{keyword}}`**
   - 영문 변수명으로 통일 (한글도 지원되지만 영문 권장)

2. **AI에게 명확한 지시**
   - "{{...}} 형식 사용 금지"
   - "실제 값으로 채워주세요"

3. **title 필드 추가**
   - `"title": "질병코드 {{keyword}} 실손24 보험금 청구"`
   - 더 이상 폴백 값("R1049 - 상세 정보")이 사용되지 않음

4. **모든 필수 필드 포함**
   - 템플릿에서 사용하는 모든 변수가 JSON에 포함됨
   - 빈 값 발생 방지

5. **배열/객체 구조 명시**
   - `usage_criteria`: 배열 `[]`
   - `detailed_causes`: 객체 배열 `[{}, {}]`
   - `insurance_table`: 객체 배열

## 사용 팁

### 변수 사용 규칙

**프롬프트에서:**
- `{{keyword}}` 사용 (시스템이 실제 키워드로 치환)

**AI 응답 JSON에서:**
- `{{keyword}}` 그대로 사용 (템플릿에서 치환됨)
- 예: `"question": "질병코드 {{keyword}}로 여러 번..."`

**템플릿에서:**
- `{{keyword}}` 사용 (권장)
- 또는 `{{disease_code}}`, `{{키워드}}` (모두 지원됨)
